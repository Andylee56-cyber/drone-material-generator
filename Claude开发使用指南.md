# Claude 开发使用指南

## 🎯 三种使用 Claude 的方式

### 方式一：在 Cursor 中使用 Claude（推荐，当前正在使用）

**优点**：已集成在 IDE 中，无需额外安装，可直接用于代码开发

#### 使用步骤：

1. **打开 Cursor**
   - 你已经在使用 Cursor 了（当前界面就是）
   - 无需额外操作

2. **使用 Claude 功能**
   - **聊天对话**：右侧边栏的聊天窗口（你现在就在用）
   - **代码补全**：输入代码时自动提示
   - **代码生成**：选中代码，按 `Ctrl+L` 或点击聊天图标
   - **代码解释**：选中代码，在聊天中提问

3. **常用快捷键**
   - `Ctrl+L`：打开/聚焦聊天窗口
   - `Ctrl+K`：内联编辑（在代码中直接修改）
   - `Ctrl+I`：代码补全提示

4. **开始开发**
   ```powershell
   # 在 Cursor 中打开你的项目
   # File → Open Folder → 选择 D:\claude-code-project
   ```

---

### 方式二：使用 Claude Desktop（独立应用）

**优点**：独立的桌面应用，功能完整，适合文档编写和对话

#### 安装步骤：

1. **下载 Claude Desktop**
   - 访问：`https://claude.ai/download`
   - 或使用国内镜像（如果有）
   - 下载 Windows 版本

2. **安装**
   - 双击安装包
   - 按照向导完成安装

3. **登录/注册**
   - 打开 Claude Desktop
   - 如果没有账户，访问 `https://claude.ai` 注册
   - 使用邮箱登录

4. **开始使用**
   - 打开应用即可开始对话
   - 可以粘贴代码、上传文件等

---

### 方式三：浏览器中使用 Claude.ai

**优点**：无需安装，随时随地访问

#### 使用步骤：

1. **访问网站**
   - 打开浏览器
   - 访问：`https://claude.ai`

2. **登录/注册**
   - 如果没有账户，点击 "Sign up" 注册
   - 使用邮箱和密码登录

3. **开始对话**
   - 在输入框中输入问题
   - 可以粘贴代码、上传文件

## 🚀 推荐工作流程（在 Cursor 中开发）

### 步骤1：打开项目

```powershell
# 在 Cursor 中
# File → Open Folder
# 选择：D:\claude-code-project
```

### 步骤2：创建开发文件

在 Cursor 中：
1. 右键项目文件夹 → `New File`
2. 创建 `index.js` 或 `app.js`

### 步骤3：使用 Claude 辅助开发

**示例：让 Claude 帮你写代码**

1. **在聊天窗口输入**：
   ```
   帮我创建一个简单的 Node.js 服务器，使用 Express 框架
   ```

2. **或者选中代码，按 `Ctrl+L`，然后提问**：
   ```
   解释这段代码的作用
   ```

3. **使用代码补全**：
   - 开始输入代码
   - Cursor 会自动提示
   - 按 `Tab` 接受建议

### 步骤4：运行和测试

```powershell
# 在 Cursor 的集成终端中（Terminal）
node index.js
```

## 🔧 配置建议

### Cursor 设置优化

1. **打开设置**
   - `File → Preferences → Settings`
   - 或按 `Ctrl+,`

2. **推荐配置**
   - 启用自动保存
   - 启用代码格式化
   - 配置字体大小（推荐 14-16px）

### 项目配置

在项目根目录创建 `.cursorrules` 文件（可选）：

```
# 项目开发规范
- 使用 ES6+ 语法
- 使用 async/await 处理异步
- 代码需要添加注释
- 使用 TypeScript 类型定义
```

## 🎓 学习资源

### 在 Cursor 中使用 Claude 的最佳实践

1. **明确提问**：描述清楚你的需求
2. **提供上下文**：选中相关代码再提问
3. **迭代优化**：根据结果继续提问改进
4. **理解代码**：不要直接复制，理解后再使用

### 常用提问模板

```
帮我创建一个 [功能描述] 的 [语言/框架] 代码
```

```
解释这段代码：[粘贴代码]
```

```
如何修复这个错误：[粘贴错误信息]
```

```
优化这段代码的性能：[选中代码]
```

## 🚨 注意事项

1. **网络连接**：Claude 需要网络连接才能工作
2. **代码审查**：生成的代码需要人工审查和测试
3. **安全性**：不要分享敏感信息（API 密钥、密码等）
4. **版本控制**：使用 Git 管理代码版本
